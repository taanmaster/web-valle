# Formatos de Alta de Proveedores

Esta carpeta contiene los formatos descargables para el proceso de alta de proveedores.

## Archivos Requeridos

### 1. formato_alta.docx
Formato de solicitud de alta para proveedores.

### 2. formato_datos_fiscales.docx
Formato de datos fiscales del proveedor.

## Instrucciones

1. Los archivos deben estar en formato Microsoft Word (.docx)
2. Los nombres de archivo deben coincidir exactamente con los especificados arriba
3. Los usuarios podrán descargar estos formatos desde su perfil de proveedor
4. Una vez llenados, los proveedores subirán estos documentos completados

## Ubicación en el Sistema

- Ruta física: `/public/front/formats/`
- URL pública: `{{ asset('front/formats/nombre_archivo.docx') }}`

## Notas

- Asegúrate de que los archivos tengan permisos de lectura
- Los formatos deben actualizarse según los requerimientos legales vigentes
- Mantén copias de respaldo de las versiones anteriores
